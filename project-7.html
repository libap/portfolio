Agenda collaboratif en python

Présentation du projet

J'ai utilisé Flask, un framework Web Python, pour : Créer une application Web d'agenda collaboratif : Cette application permet aux utilisateurs de créer des événements et de les partager avec d'autres utilisateurs. Stocker les données des événements avec SQLite3 : Pour stocker les données des événements, j'ai utilisé SQLite3, une base de données relationnelle légère et rapide. Inclure des fonctionnalités de création, modification et suppression d'événements : Les fonctionnalités de l'application incluent la création, la modification et la suppression d'événements, ainsi que la possibilité de les partager avec d'autres utilisateurs. Implémenter des fonctionnalités de filtrage et de notification : J'ai également inclus des fonctionnalités de filtrage pour afficher les événements par date, par utilisateur ou par catégorie. En outre, j'ai implémenté une fonctionnalité de notification pour informer les utilisateurs des changements apportés aux événements auxquels ils ont été invités.

Créer une application Web efficace et facile à utiliser : En utilisant Flask et SQLite3, j'ai pu créer une application Web d'agenda collaboratif efficace et facile à utiliser. Les utilisateurs peuvent facilement planifier des événements et les partager avec d'autres personnes, tout en restant informés des mises à jour de manière transparente.
